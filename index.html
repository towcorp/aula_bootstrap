Estudo de Caso #01 – Template Básico HTML com Bootstrap

O devido estudo de caso tem por finalidade levar o estudante a criar seu primeiro template, ainda que estático, com a utilização da linguagem de marcação HTML e o framework CSS Bootstrap. Dessa forma, o estudante inicia a sua jornada pela programação WEB que é um vasto campo de estudo e trabalho. O estudante deverá partir de um código básico em HTML e evoluir para um site que contenha:
um NavBar;
um header;
um body;
e um footer.
